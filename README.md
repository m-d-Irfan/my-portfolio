# Monzurul Islam — Personal Portfolio

A premium, high-performance developer portfolio website built using Next.js 15, React 19, TypeScript, Tailwind CSS, and DaisyUI.

For detailed project specifications, architectural guides, custom theme parameters, accessibility audits, and screenshots, please refer to the dedicated [documentation.md](file:///e:/My%20Portfolio/Website/documentation.md) file.

---

## 🚀 Quick Start

### 1. Installation
Install all required project dependencies:
```bash
npm install
```

### 2. Run Local Server
Launch the development server on [http://localhost:3000](http://localhost:3000) with hot-reloading:
```bash
npm run dev
```

### 3. Production Build
Compile and build the optimized production package:
```bash
npm run build
```
